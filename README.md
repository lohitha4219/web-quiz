# 🌐 Web Development Quiz

A modern and interactive **Web Development Quiz Application** built using **HTML, CSS, and JavaScript**. The application allows users to register, log in, take a timed quiz, view their score, and download their quiz results.

---

## 📌 Features

- 🔐 User Registration
- 🔑 User Login & Logout
- 📝 Web Development Quiz
- ⏱️ 20 Seconds Timer for Each Question
- 📊 Live Score Tracking
- 📈 Question Progress Bar
- ✅ Correct & Incorrect Answer Highlighting
- 🔄 Retry Quiz
- 📄 Download Quiz Result
- 🎨 Modern Responsive UI
- 📱 Mobile Friendly Design

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/lohitha4219/web-quiz.git
```

2. Open the project folder

```bash
cd web-quiz
```

3. Open `login.html` in your browser

or

Use **Live Server** in VS Code.
